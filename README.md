Customer Churn Analysis
Project Overview
This project analyzes customer churn data to identify patterns affecting customer retention. It includes data understanding, data cleaning, exploratory data analysis (EDA), customer segmentation, and advanced analysis to generate actionable business insights.

Project Tasks

Data Cleaning
Exploratory Data Analysis (EDA)
Customer Segmentation
Advanced Analysis
Business Recommendations

Key Insights
Analyzed customer churn trends.
Compared churn based on tenure, gender, payment method, and contract type.
Visualized customer behavior using charts.
Generated recommendations to improve customer retention.
